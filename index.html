HTML+CSS & Marks+Channels

  • The compound bubble and pie chart implements attributes as the data type which can be further categorized into categorical for data that is used in the pie chart and ordinal data which is used to determine the location of the bubble marker. The mark type can be area for the values within the pie chart and points when treating the pie charts as bubbles. The channel types used are color to represent the different areas within the pie chart and area/2D size of the bubbles.

  • For the bump chart, the data type being used is attributes which can be categorized as categorical for each of the different class ranks, The mark type used is links showing connection. The channel types being used are color andd position.

  • The Mind Map example shows Ideas, Categories, and Things which are all Categorical Attributes. They represent one of many abstract or concrete objects. The Marks are Points which represent each of the Mind Map's Attributes and Lines which represent the links between the Attributes. The Points Channel is the Color, indicating which type of Attribute each point is. The Lines Channel is Positions which indicates which Points are connected. The Points are items and the Lines are Links.

  • The Stacked Bar Chart example shows bars split into Categorical Attributes [A, B, C, D] and Categorical Attributes within the bars. There are Ordered Attributes that vertically measure the bars and their sections. The Marks are the Lines for each section of the bars. The Lines Channels are Color to show each of the four sections of the bar as a different category and Length which indicates the quantitative measure on the chart's y axis. The Lines are Items.
