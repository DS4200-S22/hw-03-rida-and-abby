HTML+CSS & Marks+Channels

  • The compound bubble and pie chart implements attributes as the data type which can be further categorized into categorical for data that is used in the pie chart and ordinal data which is used to determine the location of the bubble marker. The mark type can be area for the values within the pie chart and points when treating the pie charts as bubbles. The channel types used are color to represent the different areas within the pie chart and area/2D size of the bubbles.

  • For the bump chart, the data type being used is attributes which can be categorized as categorical for each of the different class ranks, The mark type used is links showing connection. The channel types being used are color andd position.
